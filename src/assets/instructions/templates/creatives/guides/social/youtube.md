# Platform Guide: YouTube

> [!NOTE]
> YouTube is centered around high-impact thumbnails and "Channel Branding" that must work on TV, Desktop, and Mobile.

## Aspect Ratios

- **Channel Banner**: 2560x1440 (16:9).
- **Safe Area (All Devices)**: 1546x423px (Centered).
- **Thumbnail**: 1280x720 (16:9).
- **Profile Pic**: 1:1 (800x800).

## Visual Strategy

### 1. The Banner "TV vs Mobile" Problem

Designing for YouTube requires a "Matryoshka" strategy:

- **Full Image (2560x1440)**: Decorative background (Visible on TV).
- **Center Banner (2560x423)**: Branding context (Visible on Desktop).
- **Safe Zone (1546x423)**: The Core Info/Logo/Schedule (Visible on ALL devices).
- **Rule**: Place ALL text and the logo inside the Safe Zone.

### 2. Thumbnail Psychology

- **High Emotion**: Use high-contrast facial expressions or emotive primary colors.
- **Large Text**: Use bold, thick fonts (e.g., Impact, Bebas Neue) with a 40/60 text-to-image ratio.
- **Rule**: Never place text in the bottom right corner (Timecode overlay).
