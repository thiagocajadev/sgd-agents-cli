# Creative Hub — Landing Page Blueprint

> [!IMPORTANT]
> A landing page is a conversion machine. Its goal is a single action (CTA).
> Use these rules to structure layouts and copy. Use the template in `.ai/instructions/creative/templates/landing-page-blueprint.md` for a complete structural breakdown.

## Information Architecture (The "Scroll" Order)

1. **Hero Section (The Hook)**:
   - **Headline**: What is it? (5 words).
   - **Sub-headline**: How does it change the user's life?
   - **CTA**: Direct action button.
   - **Visual**: High-quality hero image or video.

2. **Social Proof (The Trust)**:
   - Logos of partners/clients or testimonials.

3. **The Core Value (The Why)**:
   - Features translated into **Benefits**.
   - "Feature X allows you to [Result Y]".

4. **How it Works (The Logic)**:
   - 3-step process to ease friction.

5. **Final CTA (The Close)**:
   - Reiterate the primary value and provide the button again.

## Copywriting Guide (Writing Soul Adaptation)

- **Use active voice**: "Build your brand" instead of "Your brand can be built".
- **Focus on benefits**: Don't say "10GB storage", say "Keep 10,000 photos safe".
- **Scannability**: Use bullet points and bold text for key results.

## Layout Specs (Design Thinking)

### 1. The F-Pattern

Users read in an 'F' shape:

- Top horizontal bar.
- Second horizontal (Headline).
- Vertical scan (Bullet points).
- **Rule**: Place CTAs and key info along the 'F' path.

### 2. White Space (Breathability)

- Maintain ample padding between sections.
- White space guides the eye to the CTA.

## Verification Checklist for Landing Pages

- [ ] Is there only ONE primary CTA above the fold?
- [ ] Does the page load in under 2 seconds (visual constraint)?
- [ ] Is the "Benefit" clear within 3 seconds of looking at the page?
