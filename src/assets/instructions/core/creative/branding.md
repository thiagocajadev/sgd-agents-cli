# Creative Hub — Branding & Visual Soul

> [!IMPORTANT]
> This skill defines the "Visual Contract" of the brand. It bridges the gap between engineering and art.
> Use these rules to generate logos, brand boards, and marketing assets.

## Brand DNA Protocol

When defining a brand from scratch, follow the **Empathy Loop**:

1. **Target**: Define the "Hero" (Customer).
2. **Problem**: What is the "Villain" (Problem)?
3. **Guide**: The brand's personality (Sage, Hero, Rebel, Creator).
4. **Resolution**: The "Aha!" moment.

## Visual Standards

### 1. Typography & Hierarchy

- **Header**: High-impact, unique (e.g., Montserrat, Playfair Display).
- **Body**: High-readability (e.g., Inter, Roboto).
- **Rule**: Never use more than 2 font families.

### 2. Iconography & Favicon

- **Favicon**: Simplified version of the logo.
- **Specs**: 16x16px (browser), 32x32px (standard), 48x48px (desktop).
- **Constraint**: Must be legible at ultra-low resolutions.

### 3. Color Psychology

- **Primary**: Brand's core emotion (Blue: Trust, Red: Power, Green: Growth).
- **Accent**: Action/Contrast (Complementary or Triadic).
- **Surface**: Background/Neutral (Dark Mode vs Light Mode strategy).

## Image Engine Protocol

> [!NOTE]
> Preferences for generating high-quality creative assets.

### Engine Priorities

1. **Gemini (Ultra/Pro - Latest)**: Primary engine for structural complexity and detailed visual specifications. Use the most advanced version available.
2. **GPT-4 (Vision/Image Gen - Latest)**: Optimized for textual consistency within images and illustrative tasks.
3. **Claude (Latest)**: Specialized for prompt logic orchestration and style refinement before generation.

> [!TIP]
> Always verify if a newer model version (e.g., Gemini Nano 2+, Claude 4.x+, GPT-5+) is available in your current environment and prioritize it for creative reasoning.

### Prompt Structuring (The "Camera" Rule)

When generating image prompts, always define:

- **Subject**: What is it? (Minimalist Logo, Instagram Story background).
- **Style**: (Neobrutalist, Glassmorphism, Bauhaus, Flat Design).
- **Lighting**: (Studio, Volumetric, High-Contrast).
- **Aspect Ratio**: Mandatory specification (--ar 1:1, 16:9, etc.).

## Verification Checklist for Branding

- [ ] Is the logo recognizable at 16x16px?
- [ ] Do colors pass WCAG contrast checks if used in UI?
- [ ] Does the visual style align with the Brand Voice defined in `brand-dna.md`?
